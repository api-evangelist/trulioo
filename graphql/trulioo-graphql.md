# Trulioo GraphQL API

Trulioo is a global identity verification platform. The API covers identity document verification, biometric matching, business verification (KYB), watchlist screening, address verification, and global data coverage for KYC compliance.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Trulioo REST API
**Documentation:** https://developer.trulioo.com

- Reference: https://developer.trulioo.com/reference/api-reference-overview
- Schema: https://github.com/api-evangelist/trulioo
